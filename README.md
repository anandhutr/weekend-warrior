# CricAuction Pro

A professional cricket auction management suite built with React and Vite.

## Features

*   **Dashboard:** Overview of auction stats, budget utilization, and player distribution.
*   **Auction Arena:** Live auction interface with bidding and player shuffling.
*   **Team Management:** Manage franchises, budgets, and squads.
*   **Player Pool:** Registry of players with stats, roles, and import capabilities (CSV/Google Sheets).
*   **Persistent Database:** Uses Dexie.js (IndexedDB) to save all data locally.

## Run Locally

**Prerequisites:**  Node.js

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the app:
    ```bash
    npm run dev
    ```

## Tech Stack

*   React 19
*   Vite
*   Tailwind CSS
*   Recharts
*   Dexie.js
